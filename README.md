## English
This repository contains the core logic for the **HunTian (浑天)** 1.58-bit ternary computing architecture. By replacing traditional binary multiplication with conditional addition/subtraction and chiral masking, we achieve massive throughput on standard silicon.

### Key Features:
- **vavx3_api.h**: The definitive set of 83 sovereign instructions.
- **Multiplier-Free Arithmetic**: Efficient implementation of ternary dot products.
- **4320D Vortex Mapping**: O(1) addressing logic derived from Christoffel symbols.
